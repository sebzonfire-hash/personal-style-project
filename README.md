# Personal Style System

## Zweck

Dieses Repository dient dem systematischen Aufbau eines persönlichen Stil- und Garderobensystems.

Ziel ist keine lose Sammlung von Modeempfehlungen, sondern ein belastbares Entscheidungsmodell für:

* vorhandene Kleidungsstücke
* neue Anschaffungen
* Outfitkombinationen
* Farb- und Materialentscheidungen
* Aussortierung und Ersatz
* saisonale und anlassbezogene Garderobenplanung
