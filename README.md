# personal-style-project

## Ziel

Dieses Repository dient dem systematischen Aufbau eines persönlichen Stil- und Garderobensystems. Ziel ist keine Sammlung von Modeideen, sondern ein konsistentes Entscheidungs- und Wissenssystem.

## Projektstruktur

- docs/              Regelwerk
- wardrobe/          Kleidungsstücke
- outfits/           Outfitformeln
- assets/            Bilder und Referenzen
- recommendations/   Kauf- und Optimierungsempfehlungen

## Arbeitsweise

Neue Inhalte werden zunächst konzeptionell entwickelt und anschließend in das Projekt übernommen. Projektweite Konsistenzprüfungen erfolgen regelmäßig.
